WEB 2 - SESSION 7 - Groupproject API Documentation

GET /
Redirects you to the info page about how to use our challenges API
GET /challenges
Returns all challenges from the collection challenges
GET /challenge/:id
Returns a single challenge from the collection challenges. This method requires an ObjectId to be given along.
POST /challenge/
Creates a challenge in the database. Add data to the body of the request
PUT /challenges/:id
Update the challenge with the id provided
DELETE /challenges/:id
Delete the challenge with the id provided
